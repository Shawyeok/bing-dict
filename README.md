# bing-dict
Bing dictionary(http://cn.bing.com/dict) in command line.

## Quickstart
```shell
    $ git clone 'https://github.com/Shawyeok/bing-dict.git'
    $ sudo cp bing-dict/dic.sh /usr/local/bin/dic
    $ dic powerful
    powerful
    美['paʊrf(ə)l]，英['paʊə(r)f(ə)l]
    
    adv. 很；
    adj. 有权势的；有影响力的；强有力的；力量大的；
    网络释义： 强大的；有权的；有权威的；
```
